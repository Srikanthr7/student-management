    Overview

    EduTrack Pro provides a full-featured REST API returning JSON responses. All requests must utilize secure JSON payloads and receive standard HTTP status codes in response. The base path for all endpoints is:

    http://localhost:5000/api/v1

    All endpoints (except auth and docs) are protected by JSON Web Tokens (JWT) using the Bearer schema:

    Authorization: Bearer <your_jwt_access_token>

    Authentication

    POST /auth/token

    Obtain JWT access and refresh tokens by passing valid credentials.

    Request Body (JSON)
    {
  "username": "admin",
  "password": "Admin@123"
}
    Response (200 OK)
    {
  "status": "success",
  "message": "Token issued successfully",
  "data": {
    "access_token": "eyJhbGciOi...",
    "refresh_token": "eyJhbGciOi...",
    "user": {
      "id": 1,
      "username": "admin",
      "role": "admin"
    }
  }
}
    POST /auth/refresh

    Generate a new access token using your refresh token. Include the refresh token in the Authorization header.

    Response (200 OK)
    {
  "status": "success",
  "message": "Token refreshed",
  "data": {
    "access_token": "eyJhbGciOi..."
  }
}

    Students API

    GET /students

    Retrieve a paginated list of students. Supports filtering by search keyword and department.

    Query Parameters
    • page: Integer (default: 1)
    • per_page: Integer (default: 20)
    • search: String (filters by name, email, roll number)
    • dept_id: Integer (filters by department)
    Response (200 OK)
    {
  "status": "success",
  "data": {
    "students": [
      {
        "id": 1,
        "student_id": "ETP20260001",
        "full_name": "Arjun Sharma",
        "roll_number": "CSE2026001",
        "department": "Computer Science & Engineering",
        "year": 1,
        "semester": 1,
        "email": "arjun.sharma@student.edu",
        "cgpa": 3.7
      }
    ],
    "total": 40,
    "pages": 2,
    "current_page": 1
  }
}
    POST /students

    Create a new student record. Requires admin or teacher credentials.

    Request Body (JSON)
    {
  "full_name": "Nisha Patel",
  "roll_number": "ECE2026099",
  "department_id": 2,
  "email": "nisha.patel@student.edu",
  "year": 1,
  "semester": 1
}
    Response (201 Created)
    {
  "status": "success",
  "message": "Student created",
  "data": {
    "id": 41,
    "student_id": "ETP20260041",
    "full_name": "Nisha Patel",
    "roll_number": "ECE2026099",
    "department": "Electronics & Communication",
    "year": 1,
    "semester": 1,
    "email": "nisha.patel@student.edu"
  }
}

    Departments API

    GET /departments

    Fetch list of departments with code, HOD name, and current student count.

    Response (200 OK)
    {
  "status": "success",
  "data": [
    {
      "id": 1,
      "name": "Computer Science & Engineering",
      "code": "CSE",
      "head_of_dept": "Dr. Rajesh Kumar",
      "student_count": 8
    }
  ]
}

    Attendance API

    GET /attendance

    Query attendance records by student ID or subject ID.

    Query Parameters
    • student_id: Integer
    • subject_id: Integer
    Response (200 OK)
    {
  "status": "success",
  "data": [
    {
      "id": 12,
      "student_id": 1,
      "subject_id": 3,
      "date": "2026-06-18",
      "status": "present"
    }
  ]
}

    Marks API

    GET /marks

    Fetch student exam marks, including grade calculations and pass/fail standing.

    Query Parameters
    • student_id: Integer
    Response (200 OK)
    {
  "status": "success",
  "data": [
    {
      "id": 4,
      "student_id": 1,
      "subject": "Data Structures",
      "exam_type": "midterm",
      "marks_obtained": 82.5,
      "max_marks": 100.0,
      "percentage": 82.5,
      "grade": "A+",
      "is_pass": true,
      "semester": 1,
      "academic_year": "2024-25"
    }
  ]
}

    Statistics API

    GET /stats

    Retrieve overall database aggregates for students, departments, registrations, and subjects.

    Response (200 OK)

  "status": "success",
  "data": {
    "total_students": 40,
    "total_departments": 5,
    "new_this_month": 0,
    "total_subjects": 12
  }
}
